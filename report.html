📋 Project Overview

MusicWeb is a fully client-side music streaming platform built with semantic HTML5, CSS3, and vanilla JavaScript. It provides a modern, responsive interface for browsing, playing, and managing music — all running directly in the browser without any server-side dependencies. Favorites and playlists persist across sessions using the Web Storage API.

✨ Key Features

  • Music player with play, pause, previous, next, and seek controls
  • Progress bar with click-to-seek and real-time time display
  • Volume control with persistent state across page navigation
  • Favorites system — heart any track and view your collection on the Favorites page
  • Playlist management — create, rename, and delete custom playlists (stored in localStorage)
  • Live search across all 15 tracks by title, artist, or album
  • Browse page with categories, new releases, and editor's picks
  • Settings page with audio quality, notifications, and display preferences
  • Keyboard shortcuts: Space (play/pause), N (next), P (previous), F (favorite), Alt+← → (navigate pages)
  • Ripple animation effect on all interactive buttons
  • Responsive design for mobile, tablet, and desktop screens
  • Accessible markup with ARIA labels, roles, and keyboard navigation
  • Smooth CSS animations and page transitions
  • Player state persists across page navigation via sessionStorage

📄 Application Pages

  • Home (index.html) — Hero section, featured playlists, trending tracks, top artists
  • Browse (browse.html) — Live search bar, music categories, new releases, editor's picks
  • Playlists (playlists.html) — Create, edit, and delete personal playlists with stats
  • Favorites (favorites.html) — Dynamic list of all favorited tracks with playback
  • Settings (settings.html) — Audio quality, notifications, display preferences, keyboard shortcuts reference
  • Report (report.html) — This page — project overview, features, tech stack, live stats

🛠️ Technology Stack

Built entirely with web standards — no frameworks, no bundlers, no dependencies.

HTML5 CSS3 Vanilla JavaScript (ES2022) CSS Custom Properties CSS Grid & Flexbox Web Storage API Google Fonts (Vazirmatn) CSS Animations & Transitions ARIA Accessibility Responsive Design
